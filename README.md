# Flower-Shaped L Keychain

## Project Overview

This project is a custom flower-shaped keychain designed in Onshape.  
The design was inspired by a simple floral keychain reference and was recreated as a 3D-printable model with my own modifications.

The keychain features a capital letter **L** in the center, representing my name, **Lujain**. A circular hole was added to one side of the flower so the model can be attached to a keyring.

## Design Features

- Flower-shaped outer profile
- Capital letter **L** in the center
- Circular keyring hole
- Rounded and smooth edges
- Single-piece 3D-printable design
- Simple geometry with minimal details
- Suitable for exporting as an STL file

## Design Process

The model was created in Onshape using sketches and extrusion features.  
The flower outline, keyring hole, and letter were created in separate sketches, then combined using extrusion and removal operations to produce the final model.

## Inspiration

The design was inspired by a minimalist flower-shaped keychain.
